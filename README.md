# 🛍️ buyForever - Premium E-Commerce Wardrobe Experience

TrendFlow ek modern, ultra-responsive aur visually striking clothing storefront hai jise khas taur par Indian audience ke liye design kiya gaya hai. Yeh website ek minimalist fintech layout ko ek high-energy streetwear aesthetic me transform karti hai.

---

## 🎨 Theme & Visual Identity

Website ka UI ek vibrant aur energetic color palette par based hai jo glassmorphism components ke sath milkar ek premium look deta hai.

* **Background Gradient:**
    ```css
    background: linear-gradient(90deg, hsla(280, 95%, 57%, 1) 0%, hsla(193, 90%, 55%, 1) 100%);
    ```
* **Visual Elements:** Background me dynamic floating clothing icons (`👕`, `👖`, `👗`, `👟`) hain jo store ke content ko enhance karte hain bina design ko cluttered banaye.
* **Design Style:** Translucent Glassmorphic containers (blur effects + subtle white borders) for Login & Registration forms.

---

## ✨ Features

* **⚡ All the drip, none of the hype:** Faltu ke clutter ke bina sirf premium aur curated wardrobe collections par focus.
* **🛒 Trend Tracking & Tags:** Users apni clothing preferences (Oversized, Casuals, Winterwear) ko custom tags se filter kar sakte hain.
* **🔒 Secure & Private Checkout:** UPI, Card aur Cash options ke sath secure logging aur zero third-party trackers.
* **📱 Universal Responsiveness:** Phone, tablet, aur desktop par flawless experience.
* **🔑 Complete Auth Flow:** Sync kiya hua matching Login aur Signup pages jo pure user flow ko smooth banate hain.
